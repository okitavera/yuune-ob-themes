# yuune-ob-themes
My openbox theme collection :D

## Installation

clone this repo into your local storage:

    $ git clone https://github.com/yuune/yuune-ob-themes.git
    $ cd yuune-ob-themes

copy the theme file to your tint2 config folder:

    $ cp -avf [themename] ~/.themes/
